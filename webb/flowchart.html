Alur Proses Analisis Pengaruh Radiasi terhadap Pertumbuhan Rumput Laut

Diagram Alur Proses

Penjelasan Flowchart
Flowchart di atas menggambarkan alur kerja lengkap dari proses analisis data pertumbuhan rumput laut berdasarkan radiasi cahaya. Proses dimulai dengan input data dari berbagai tabel (Lingkungan, Pengukuran, Kebutuhan, Lokasi, dan Nutrien), kemudian dilakukan validasi data untuk memastikan kelengkapan dan validitas. Setelah data valid, dilakukan penggabungan data menggunakan Power Query BI, diikuti dengan perhitungan growth rate dan pengkategorian radiasi. Analisis statistik dilakukan untuk melihat hubungan antara variabel, dan berdasarkan hasil analisis, sistem akan menghasilkan output berupa hipotesis terbukti atau tidak terbukti, yang kemudian divisualisasikan dalam dashboard interaktif.
